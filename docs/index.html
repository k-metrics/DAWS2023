環境構築

本資料では『統計ソフト「R」超入門』逸見功著で利用するR/R Commander環境を構築する方法を紹介しています。Windows10/11環境のみが対象です。他のOSを利用したい場合はCRANにてインストール方法を確認してください。

環境構築は大きく以下の手順に分かれます

  1. Rのインストール
  2. R Commanderのインストール
  3. 動作確認
ユーザーアカウント名について

Rをインストールする際に利用するWinodwsのユーザアカウント名は日本語および空白を含まず数字から始まらないアカウントを使用してください。日本語や空白などを含んだアカウント名や数字から始まるアカウント名を使用するとRがパスを処理できず起動しなかったり正しく動作しない場合があります。

　

Rのインストール

ダウンロード

  1. CRANからRの最新版をダウンロードします
  2. ダウンロードしたファイルのウィルスチェックをします
Tip: Version

書籍と同じバージョンを利用したい場合はこちらからR4.2.2をダウンロードしてください。

インストール

Rのインストールではダイアログでの問い合わせに対して基本的にデフォルトを選択します。ただし、R Commnderを使いやすくするために起動時オプションの選択だけはデフォルトから変更します。

  1. ダウンロードしたファイルを実行
  2. インストール言語を選択
  3. ライセンスの確認
  4. インストール先の指定
  5. コンポーネントの選択
  6. 起動時オプション
    • 「はい(カスタマイズする)」を選択
    1. 表示モード
      • 「SDI(複数のウィンドウを使用して表示)」を選択
    2. ヘルプの表示方法
      • 「HTMLヘルプ」を選択
  7. スタートメニューフォルダーの指定
  8. 追加タスクの選択
    • 「アイコンの追加」はお好みで選択
  9. インストール状況（プログレス表示）
  10. 完了
Tip: MDI or SDI

R Commanderを利用する場合は、操作ウィンドウやグラフ表示に個々のウィンドウを割り当てるSDIモードが便利です。インストール時にMDIを選択してた場合でもR起動後にオプションで変更できます。

　

起動確認

スタートメニューから［R］-［R 4.x.x］を実行し下図のようなウィンドウが表示されることを確認します。

　

R Commanderのインストール

インストール

R CommanderのインストールはRから行います。

  1. Rのメニューから［パッケージ］-［パッケージのインストール…］を実行
  2. パッケージをダウンロードするミラーサイトを選択して［OK］をクリック
    • デフォルトの「0-Cloud[https]」を選択
  3. パッケージ一覧から「Rcmdr」を選択して［OK］をクリック
  4. パーソナルライブラリの利用確認
    • 初回のみで［はい(Y)］をクリック
  5. インストール先の確認
    • 初回のみで［はい(Y)］をクリック
  6. パッケージダウンロードのプログレスが複数表示
    • なにもしない
  7. R Commanderインストール完了

　

起動確認

R Commanderが起動することを確認します。

  1. Rのメニューから［パッケージ］-［パッケージの読み込み…］を実行
  2. 「Rcmdr」を選択
  3. 不足パッケージのインストール
    • 初回のみ
  4. ダウンロード先の選択
    • デフォルトの「CRAN」を選択
  5. R Commander起動

　

動作確認

以下はR Commanderウィンドウでの操作になります。

  1. メニューバーの下にある［データセット:］の部分が「<アクティブデータセットなし>」になっていることを確認
  2. メニューから［データ］-［パッケージ内のデータ］-［アタッチされたパッケージからデータセットを読み込む…］を実行
  3. ［データセット名を入力:］欄に「iris」と入力し［OK］をクリック
  4. メニューバーの下にある［データセット:］の部分が「iris」になっていることを確認
  5. メニューから［グラフ］-［箱ひげ図…］を実行
  6. ［変数］の「Petal.Length」をクリックし［OK］をクリック
  7. 別ウィンドウで箱ひげ図が描かれることを確認
Tip: Version

「アクティブデータセット」はR Commander特有の概念です。R単体でも同様のことはできますが変数の参照間違いにつながりやすいので一般的には使用しません。
